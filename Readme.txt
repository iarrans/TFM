REQUISITOS
⁃ Conexión a internet
- 4 mandos (1 por jugador) o 3 mandos y 1 teclado (solo 1 puede jugar con teclado)
⁃ Parsec 
        - todos los jugadores deben tenerlo instalado
        - todos los jugadores deben instalar el driver adicional de Parsec VUSB con versión 0.2.8.0 o posterior.
        - el host debe configurar lo siguiente: Settings > Host Settings > Parsec Virtual USB Gamepad (Enabled) y Virtual Gamepad Type (Xbox 360)
⁃ Build del juego
        - debe tenerla el host


INSTRUCCIONES
1. El jugador que tenga la build (host) debe iniciar una sesión en Parsec y compartir el enlace con el resto de jugadores. Cuando estos se vayan uniendo, los tiene que aceptar.
2. El host debe ejecutar el archivo .exe
3. El host debe iniciar la partida. Todos los jugadores que estén conectados a la sesión de Parsec podrán unirse al juego.


CONTROLES EN TECLADO
⁃ Movimiento con WASD
⁃ Rotación con R
⁃ Pausa con ESC


CONTROLES EN MANDO
⁃ Movimiento con joystick izquierdo
⁃ Rotación con X (PS5) o A (XBOX)
⁃ Pausa con botón Options/Settings/Pause
